Call

implements Future<V>
com.magnet.android.mms.async.Call<T>

Class Overview

This interface represents an asynchronous invocation to a controller. An instance of the Call is typically returned by a Controller method call. Calling dispose() or dispose(boolean) is optional, but these methods allow developers to delete the cached response immediately.

public abstract boolean cancel (boolean mayInterruptIfRunning)

Cancel a queued or executing call. If the call has been disposed, completed, cancelled, or is unable to cancel, it will return false. Upon successful completion, this call object will be disposed. There is no guarantee that the call can actually be cancelled; the system makes its best effort.

Parameters
mayInterruptIfRunning Set to true to permit interruption if the call is running; set to false otherwise.
Returns
  • true if cancellation is submitted; false if unable to cancel.

public abstract boolean dispose ()

Dispose this Call without clearing its result from cache. The call must be in a SUCCESS, FAILED or TIMEDOUT state. All resources used by this Call will be released. To dispose a queued or executing call, it must be cancelled first. This method is equivalent to dispose(boolean) with false.

Returns
  • true for success; false if in an illegal state.

public abstract boolean dispose (boolean clearResult)

Dispose this Call and optionally clearing its result from cache. The call must be in SUCCESS, FAILED or TIMEDOUT state. All resources used by this Call will be released. The clearResult allows the caller to remove its result from the cache immediately; it is useful to any infrequent calls with large result data. But caution should be taken since it may delete the cached result for other concurrent similar calls.

Parameters
clearResult true to remove its result if exists.
Returns
  • true for success; false if in an illegal state.

public abstract boolean isCancelled ()

Indicates whether this task was cancelled before it completed normally. If the call has been disposed, it will return true.

Returns

public abstract boolean isDone ()

Indicates whether this task completed. Completion may be due to normal termination, an exception, or cancellation; in all of these cases, this method will return true. If the call has been disposed, it will return true.

Returns
  • true if this task completed or the call has been disposed; false otherwise.

public abstract boolean resend ()

Resend this request if its state is FAILED or TIMEDOUT. This operation is invalid if this Call instance has been disposed.

Returns
  • true if it is resubmitted; false if this call has an invalid state.
